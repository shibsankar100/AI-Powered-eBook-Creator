# eBookCreator

eBookCreator is an AI-powered ebook creation platform that helps users generate, write, manage, and export ebooks efficiently. The application combines modern web technologies with Google Gemini AI to simplify the ebook creation process.

## Features

### User Authentication

* Secure user registration and login using JWT authentication.

### AI Outline Generator

* Generate complete ebook outlines with chapter titles and descriptions using AI.

### AI Chapter Writer

* Create detailed chapter content instantly with Google Gemini AI.

### Markdown Editor

* Write and format ebook content using a clean and intuitive Markdown editor.

### Real-Time Preview

* Switch between edit and preview modes to view formatted content instantly.

### Drag & Drop Chapters

* Reorder chapters easily for better ebook organization.

### Book Details Management

* Update book title, subtitle, author information, and upload custom cover images.

### Export Options

* Download ebooks in PDF and DOCX (Word) formats.

### Dashboard Management

* View, edit, and delete ebooks from a professional dashboard interface.

### Responsive UI

* Mobile-friendly and modern user interface built with Tailwind CSS.

## Technology Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* Lucide React

### Backend

* Node.js
* Express.js
* MongoDB

### Authentication

* JWT (JSON Web Token)

### AI Integration

* Google Gemini AI
